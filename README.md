# HTML5Game
WIP Action/Adventure game in HTML5

## Completed Tasks
* Movement
* Hero - Player Controlled
* AI (NPC and Monster)
* Random
* Follow
* Flee
* Collision
* Display Messages/Dialogue
* Repeat Last Dialogue
* Random Dialogue
* Dialogue choices
* Treasure Chests
* Inventory
* Shops
* Menu
* Auto filling tileset. (Corners and edges are programmatically filled in based on neighboring tiles.)
